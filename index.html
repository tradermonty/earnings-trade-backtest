🚀 クイックスタート

# 過去6ヶ月のバックテスト python earnings_backtest.py --start_date 2024-01-01 --end_date 2024-06-30 # 保守的な設定 python earnings_backtest.py --stop_loss 4 --position_size 4 --sp500_only # アグレッシブな設定 python earnings_backtest.py --stop_loss 8 --position_size 8 --no_mid_small_only

📋 戦略概要

  • エントリー条件: 決算サプライズ≥5%、ギャップアップ≥0%
  • リスク管理: 6%ストップロス、21日移動平均トレーリング
  • ポジション管理: 資金の6%、最大10ポジション
  • 対象銘柄: 中小型株（時価総額$300M-$10B）

🚀 Earnings Swing Trading Strategy Backtest

Python 3.11+ EODHD API Mid/Small Cap

A comprehensive backtesting system for earnings-based swing trading strategies, specialized for mid and small-cap stocks using real-time data from EODHD API.

🎯 Key Features

📊 Advanced Analysis Engine

Comprehensive strategy analysis integrating earnings surprises, technical analysis, and risk management

🔍 Flexible Filtering

Screening with diverse conditions including market cap, volume, and price movements

📈 Detailed Reports

Generate beautiful HTML reports and CSV data exports

🛡️ Risk Management

Automated control of stop-loss, position sizing, and portfolio diversification

🚀 Quick Start

# Clone repository git clone https://github.com/tradermonty/earnings-trade-backtest.git cd earnings-trade-backtest # Create and activate virtual environment python3.11 -m venv venv source venv/bin/activate # macOS/Linux # Install dependencies pip install -r requirements.txt # Basic execution python earnings_backtest.py

💡 Usage Examples

# 6-month backtest python earnings_backtest.py --start_date 2024-01-01 --end_date 2024-06-30 # Conservative setup python earnings_backtest.py --stop_loss 4 --position_size 4 --sp500_only # Aggressive setup python earnings_backtest.py --stop_loss 8 --position_size 8 --no_mid_small_only

📋 Strategy Overview

  • Entry Conditions: Earnings surprise ≥5%, gap up ≥0%
  • Risk Management: 6% stop-loss, 21-day MA trailing
  • Position Management: 6% of capital, max 10 positions
  • Target Stocks: Mid/small-cap ($300M-$10B market cap)
